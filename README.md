# twenty

Welcome to Twenty!

## Setup & Development

### Frontend

```
cd front && npm install
npm run start
```

Browse: `localhost:3000`

### Backend

```
cd server && npm install
npm run start:dev
```

Browse: `localhost:5000`

## Tests

### Frontend

```
npm run test
```

### Backend

```
npm run test
```

Browse: `localhost:5000`

## Production

```
cd front && npm run build
cd ../server && npm run build
```
